About Me

Hello and welcome to my website! My name is Youssef Abdelhamid, I am 19 years old and am currently studying Computer Science and Interactive Media at New York University Abu Dhabi. Before moving to the United Arab Emirates for university, I lived in Egypt for my whole life. I started to learn coding on my own when I was sixteen, eventually landing several freelancing jobs over the past couple of years. The inception of this website came as a final project to the Communications Lab class at NYUAD where I built websites to host several artistic projects both alone and in groups.

Technologies I use:

  • CSS
  • Bootstrap
  • Java
  • Processing
  • Adobe Premier
  • JavaScript
  • HTML
  • Python
  • C++
  • Adobe XD
